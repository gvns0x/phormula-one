# F1 Car 3D Model

Place an F1 car model as `f1-car.glb` in this directory.

**Download sources (GLB/glTF format):**
- [FetchCFD F1 2026](https://fetchcfd.com/threeDViewGltf/4846-f1-2026-car-3d-model-in-3D) - CC Attribution
- [Sketchfab Formula 1 Car](https://sketchfab.com/3d-models/formula-1-car-e89589184eac42c08028db5cba3f6499) - Download as glTF
- [Sketchfab F1 2022](https://sketchfab.com/3d-models/f1-2022-f5b9ca6095124bf796934c150d178a02) - Lightweight option

If no model is present, the game uses a procedural fallback car.
